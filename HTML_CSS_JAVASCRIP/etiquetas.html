Estructura básica: : Define el tipo de documento y la versión de HTML utilizada. : Define el elemento raíz del documento HTML. : Contiene metadatos y enlaces a archivos externos. : Define el título del documento. <body>: Contiene el contenido visible del documento. Elementos de texto: <p>: Define un párrafo. <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Define encabezados de diferentes niveles. <strong> o <b>: Define texto en negrita. <em> o <i>: Define texto en cursiva. <u>: Define texto subrayado. <br>: Inserta un salto de línea. <hr>: Inserta una línea horizontal. Elementos de lista: <ul>: Define una lista desordenada. <ol>: Define una lista ordenada. <li>: Define un elemento de lista. Elementos de enlace: <a>: Define un enlace. <href>: Especifica la URL de destino del enlace. Elementos de imagen: <img>: Inserta una imagen. <src>: Especifica la ruta de la imagen. <alt>: Proporciona un texto alternativo para la imagen. Elementos de formulario: <form>: Define un formulario. <input>: Define un campo de entrada. <button>: Define un botón. <select>: Define una lista desplegable. <option>: Define una opción dentro de una lista desplegable. <textarea>: Define un área de texto. Elementos de tabla: <table>: Define una tabla. <tr>: Define una fila en una tabla. <td>: Define una celda de datos en una tabla. <th>: Define una celda de encabezado en una tabla. Elementos de estilo y diseño: <div>: Define una sección genérica o un contenedor. <span>: Define una sección en línea genérica o un contenedor. <style>: Define estilos CSS en línea. <link>: Vincula un archivo CSS externo. <script>: Define un script JavaScript.
